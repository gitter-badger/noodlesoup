0.2.0:
- Added drafts system:
  - When a writer posts a new post, it is added to the drafts
  - From there, Proofreaders (PRs) review the post and make
    final adjustments
  - When they are done, they submit and the post becomes readable
    for all
- Added roles system
- Administrators can edit other users' roles
- Only writers can access the posting functionality
- Only PRs can access the drafts functionality
